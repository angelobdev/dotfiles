## Installation

**Install single package:**

```sh
git clone https://github.com/angelobdev/dotfiles.git
cd dotfiles
stow <package name>
```

**Install all packages:**

```sh
git clone https://github.com/angelobdev/dotfiles.git
cd dotfiles
stow */
```

**_NOTE:_** GNU stow is required.
